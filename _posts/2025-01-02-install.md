# Server

启动server
```yaml
go run main.go
```
`-h` 获取更多帮助
```bash
Karlin Framework [Version: 1.1.1]

Usage:
  karlin [flags]
  karlin [command]

Available Commands:
  completion  generate the autocompletion script for the specified shell
  help        Help about any command
  operator    Do operation with operators and exit
  version     Print version and exit

Flags:
  -c, --config string   config file path (default "pkg/config/server.yaml")
  -h, --help            help for karlin

Use "karlin [command] --help" for more information about a command.
```
初次使用时将根据`/backend/server/pkg/config/server.yaml`初始化用户并创建数据库
```bash
server:
  host: 127.0.0.1
  file:
    port: 9000
    pattern: /files/downloads/
  db:
    name: team.db
    root: Lockly
    passwd: 3ty1r86bn
```

# Client
从release页面获取对应的二进制文件执行即可

若需自行编译可使用makefile(只适配了windows):
``` bash
make windows-amd64
```
客户端的编译依赖[Wails](https://wails.io/zh-Hans/docs/reference/runtime/dialog/#%E6%89%93%E5%BC%80%E9%80%89%E6%8B%A9%E6%96%87%E4%BB%B6%E5%AF%B9%E8%AF%9D%E6%A1%86%E5%8F%82%E6%95%B0%E9%80%89%E9%A1%B9), 但同时使用`garble`, `upx`(需要本地环境中存在)进行混淆和压缩:
``` bash
wails build -obfuscated -garbleargs "-literals -tiny -seed=myrandomseed" -upx -upxflags "-9" -ldflags "-w -s"
```
