[Karlin](https://github.com/BKLockly/KarlinC2)是一个跨平台的后渗透利用红队框架，旨在为操作员提供协作和用户友好的界面。整体使用[Wails](https://wails.io/zh-Hans/)，后端使用 [Golang](https://golang.google.cn/)，前端用 [Vue.js](https://cn.vuejs.org/) + [Naive UI](https://www.naiveui.com/zh-CN/os-theme) 编写。
