[Karlin](https://github.com/BKLockly/KarlinC2)是一种后渗透利用的命令与控制（C2）工具，利用多种通信方式来控制已被攻破的机器。整体使用[Wails](https://wails.io/zh-Hans/), 后端使用 [Golang](https://golang.google.cn/)，前端用 [Vue.js](https://cn.vuejs.org/) + [Naive UI](https://www.naiveui.com/zh-CN/os-theme) 编写。
